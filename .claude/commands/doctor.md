description: Verifica prerequisiti per AgentFlow
---
Verifica che questi tool siano installati e funzionanti:
- claude CLI: esegui `claude --version`
- git: esegui `git --version`
- gh (GitHub CLI): esegui `gh --version`
- Node.js: esegui `node --version`
- npm: esegui `npm --version`

Per ogni check mostra: ✓ nome (versione) oppure ✗ nome — non trovato, installa con [istruzione].
